# Alfanumrik Web (Workflow-First Frontend)

This folder is reserved for the Next.js app used by students, teachers, parents, and admins.

## Product UX principles

- Build workflows first (doubt solving, practice, reports, interventions)
- Avoid generic free-form chat as the primary UI
- Send structured request payloads to backend orchestrator
- Render concise, actionable outputs with mastery-aware next steps

## Planned modules

- Student learning workspace
- Teacher copilot workspace + worksheet builder
- Parent progress snapshot
- Institution performance dashboards
